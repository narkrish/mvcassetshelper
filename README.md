## MVC Assets Helper

Lightweight assets helper for ASP.NET MVC, nuget package.

## Package and deploy

```
$ dotnet restore
$ dotnet pack
$ dotnet nuget push bin/debug/*.nupkg -s https://www.nuget.org/api/v2/package -k <apikey>
```
